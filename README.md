# ai-for-sustainability
Geospatial data processing pipeline for Delhi NCR using satellite imagery. Includes spatial filtering, raster label creation, and dataset preparation for machine learning. AI for Sustainability – Geospatial Data Processing Pipeline

This repository contains a geospatial data processing pipeline developed as part of a technical internship task.

The workflow includes:
- Satellite image filtering
- Spatial clipping using GeoJSON boundaries
- Raster tile generation and land cover labeling
- Dataset preparation for machine learning tasks

The pipeline demonstrates how remote sensing data can be processed to support AI-driven sustainability applications such as environmental monitoring and agricultural analysis.
